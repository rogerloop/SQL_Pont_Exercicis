# SQL BootCamp - VS Code Extension Recommendations

Install these extensions to enhance your SQL development experience:

## Required Extensions
- **SQL Tools** (mtxr.sqltools) - Manage databases directly from VS Code
- **Better Comments** (aaron-bond.better-comments) - Highlight different comment types

## Recommended Extensions
- **Markdown Preview Enhanced** (shd101wyy.markdown-preview-enhanced) - Better markdown support
- **GitLens** (eamodio.gitlens) - Track code changes
- **Prettier** (esbenp.prettier-vscode) - Code formatting
